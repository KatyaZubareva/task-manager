# Task-manager

Task-manager — это современное веб-приложение для управления личными и командными задачами. С его помощью вы можете:

- Создавать и редактировать задачи
- Отслеживать прогресс с помощью статусов: Pending, In Progress, Review, Done, Blocked
- Удалять и отмечать задачи как выполненные
- Работать с фронтендом на Vue.js и бэкендом на Python

Приложение помогает организовать рабочий процесс и повышает продуктивность.
